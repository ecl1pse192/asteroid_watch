NEO Observer ☄️
Информационно-аналитическая система мониторинга потенциально опасных астероидов на основе данных NASA.
Ссылка на рабочий проект:(будет добавлена позже)

Технологии
Python 3.12
Django 5.0
NASA NeoWs API
Bootstrap 5

Запуск
1. `python -m venv venv`
2. `pip install -r requirements.txt`
3. `python manage.py migrate`
4. `python manage.py runserver`
